# HJS-KXML2
> KXML2 pull parser classes for the Hubrisjs javascript framework.

KXML2 implementation classes: KXmlParser, KXmlSerializer, KNode, KElement, KDocument.

## Installation

Node:

```sh
npm install hjs-kxml2 --save
```
